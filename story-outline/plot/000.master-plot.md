# 主线：在四秒静音中找回自己的声音

## Promise

玩家先破解一张异常电话网络，再发现自己就是网络中被删除的十九号接线员Evelyn Vale。每条支线都会回答一个具体问题、改变一个独立人物的处境，并提供一项能在最终行动中使用的证据或关系。最终谜底不是“这一切都是主角的幻想”，而是主角曾参与创造一套伤害他人的制度，现在必须决定如何承担和终止它。

主要类型是叙事谜案，人物戏剧为第二驱动力。故事始终以Vale为单一主角；玩家通常不会比她提前知道真相，但独立人物会提供彼此不完全一致的现实证词。

## Number Spine

主线交接固定为：`946 0264` → `871 4000` → `871 4003 / 871 4019` → `301 1968 / 794 1966` → `871 4036` → `871 4127`。其中`735 0194`与`871 4227`只承载梅芙和彼得的人物支线。号码可以提前发现，证据与权限不能被跳过。

## Pressure

### 外部压力

阿利斯泰尔·雷恩逐步识别MCE-0里的来电者。他需要Vale恢复足够记忆来补全礼仪协议，又必须阻止她恢复自主选择。Meridian通过天气服务、工作考核、自动转接和礼仪等级施压，但不能读心或瞬间控制人。

### 内部压力

Vale越接近身份真相，越必须面对自己的设计确实伤害过彼得、梅芙和其他陌生人。她擅长使用Meridian话术，因此最有效的潜入手段也最可能把她送回“入职”结局。

### 关系压力

伦纳德、多萝西、彼得和梅芙都有独立目标。他们会因为玩家此前的态度提供帮助、拒绝合作或保护自己。主角不能把他们当作自己记忆的延伸，也不能只收集线索而不承担人物后果。

### 结构压力

一个夜班不要求揭示全部真相。失败结局会归档号码、线索和后果，下一夜班从新的知识状态开始。主线预计用三至六次夜班完成，但每次夜班本身仍需有明确局部目标和结局。

夜班之间不倒转时间。交班记录卷让Vale保留记忆，独立人物因真实经历而自然记得，Wren则会根据日志适应玩家手段。每次重开都增加关系或制度成本，不能作为撤销责任的按钮。

## Turns

### 第一幕：相信眼前的雨

- **开场状态：** 没有姓名的玩家在MCE-0电话亭里醒来，只知道窗外下雨、紧急服务找不到这座电话亭。
- **诱发事件：** 梅芙主动来电，询问这里是否也在下雨；天气服务随后要求玩家给出标准答案。
- **第一选择：** 玩家可以尊重梅芙的观察、服从录音，或追问谁在收集答案。这个选择建立“相信真实的人还是标准声音”的核心矛盾。
- **第一转折：** 天气卡、黑金广告和主动服务共同指向Meridian。玩家明白异常不是单个错号，决定主动调查号码之间的联系。
- **可达局部结局：** “天气很好”展示失去判断；“号码磨损”惩罚无目标的暴力试拨；普通挂断保留第一批号码和梅芙状态。

### 第二幕上半：号码后面是真实的人

- **扩张：** 玩家接触疲惫推销员彼得、投诉技术员伦纳德和失物招领管理员多萝西。三人分别从工作、技术和档案角度证明Meridian是现实机构。
- **人物测试：** 彼得需要被当作受害者而非工具；伦纳德只接受可验证故障；多萝西只接受身份凭据。玩家若只追问号码，会削弱后期合作。
- **错误捷径：** 午夜广播可以提前给出中央交换机号码，但没有投诉权限的直拨只会进入永远转接。这让玩家明确理解“知道号码不等于拥有路径”。
- **线索收束：** 被划铭牌、彼得的警告和十九号故障回执把分散路线统一到871 4019。
- **中点揭示：** 伦纳德读出回执签名，阿利斯泰尔第一次直接称呼“Evelyn”。主角确认E. Vale不是待寻找的另一名接线员，而是自己。

### 第二幕下半：受害者也是设计者

- **新目标：** Vale需要从多萝西处认领姓名，并取得能够证明身份的外部证据。
- **反转：** Vale原声磁带承认自己参与建造礼仪协议。她后来反抗是真的，她早期推动实验也是真的。
- **关系破裂：** 伦纳德质问她为何直到实验伤及同事才反对；彼得发现自己的训练来自她的设计。若玩家此前利用他们，二人会拒绝最终协助。
- **阿利斯泰尔的计划：** 他并非单纯阻止调查，而是故意让Vale接近记忆，希望她补回协议缺失的安全规则和身份模型。
- **最低点：** Vale认领姓名的同时暴露完整线路特征，阿利斯泰尔封锁投诉台；梅芙或彼得可能因玩家早期选择被招募。主角知道了自己是谁，却看似失去所有可用关系。
- **失败的价值：** “入职”结局让玩家知道Meridian如何使用Vale；下一夜班可借此解锁“假代表”路线，但入职本身仍是失败而非必要美化。

### 第三幕：四秒里作出的选择

- **重组：** 玩家用医院便笺证明四秒时间差，用十九号回执恢复投诉资格，用失物认领卡确认姓名，用公用电话钥匙打开维护面板。
- **人物汇流：** 伦纳德提供人工窗口；多萝西确认凭据；彼得选择警告或上报；梅芙提供现实时间记录。每个人只完成自己有能力完成的一部分。
- **最终对峙：** 阿利斯泰尔以Vale的责任、盟友安全和恢复完整记忆为筹码，要求她重新入职。玩家必须先提交真实故障、保持四秒静音，再触碰维护面板上的线路。
- **高潮选择：** 切断礼仪训练线；用代表身份把招募来电改接到无人接听的清晨；接管夜班总机先行警告受害者；或因服从、犹豫和错误判断进入失败结局。

### 结局结构

- **断线 / 主成功：** Vale切断训练线，整座城市安静四秒。代价是未取回的网络记忆随线路消失；她保住姓名，却不能再拼回一个毫无缺口的过去。
- **假代表 / 部分成功：** Vale利用见过“入职”的经验骗过验证并改接来电。Meridian暂时失效，阿利斯泰尔和制度本身仍可能恢复。
- **夜班接线员 / 隐藏成功：** Vale接管总机，保留网络来保护新来电者。她救下更多人，但必须继续留在MCE-0，无法返回普通生活。
- **入职 / 失败：** 阿利斯泰尔补全Vale并把她重新变成代表；也为后续反骗留下知识。
- **永远转接 / 失败：** 玩家知道路线却回避作出责任选择，被困在自我转接中。
- **号码磨损 / 失败：** 玩家用无目标试拨代替理解，最终磨掉号码和可验证证据。
- **天气很好 / 失败：** 玩家接受标准答案高于亲眼所见，最后用自己的声音否认雨。

### Setups, Reveals, Payoffs

| Setup | Reveal | Payoff |
| --- | --- | --- |
| 天气服务只提问、不预报 | Meridian在校准回答而非天气 | 梅芙线与“天气很好”结局 |
| 多次录音停顿四秒 | 投诉线路保留人工纠错窗口 | 最终静音与切线操作 |
| 被划掉的十九号和E. Vale签名 | 主角就是Evelyn Vale | 身份认领与总机权限 |
| 黑金材料渗入公共服务 | Meridian是现实机构和承包网络 | 玩家可视觉辨认被接管线路 |
| 空白姓名认领卡 | Vale主动设置外部证据门槛 | 认领姓名而不让系统自动补全 |
| 彼得与梅芙的个人记录 | 来电者拥有主角之外的生活 | 最终联盟或具体损失 |
| 原声磁带保留旧口音 | Vale既参与项目又试图叫停 | 承担责任后的高潮选择 |
| 失物招领中的公用电话钥匙 | MCE-0有实体维护面板 | 打开投诉线与训练线接点 |
