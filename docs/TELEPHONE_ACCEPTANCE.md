# Telephone 完整验收报告

验收日期：2026-07-17

验收分支：`codex/telephone-production`

## 结论

Telephone 已按《Telephone 完整开发计划》实现为完整网页叙事解谜游戏，而非玩法原型。产品包含精制 2D 雨夜电话亭、真实转盘与听筒交互、事件驱动剧情引擎、可视化剧情后台、跨周目档案、7 个可达结局、响应式适配、实时合成音效和本地持久化。

默认剧情通过结构校验且没有错误或提醒。唯一剧情源为 `src/story/telephone.rules.json`。

| 剧情 / 系统 / 场景 / 结局节点 | 40 | 12–18 个第一版骨架 |

## 计划逐项验收

### 前台交互

- [x] 听筒支持点击提起、拖动、线长约束、拖回挂钩和键盘挂断。
- [x] 转盘使用孔位、角度、顺时针行程、挡片阈值与机械回弹建模；`0` 行程最长。
- [x] 拖动不足不确认数字，并播放机械失败反馈。
- [x] 七位号码和 `999` 紧急号码自动尝试接通；未知号码与已登记但无剧情号码分别进入空号和忙音流程。
- [x] 待机主动来电可接听或错过，未接状态进入通话记录。
- [x] 云朵回应支持条件、隐藏选项、flags、effects、超时和跨周目条件。
- [x] LCD 显示号码、拨号 / 接通状态、异常文案与通话时长。
- [x] 拨号、选择和整通电话均有告警与到期处理；新操作会安全作废旧计时器。
- [x] 6 个场景热点支持 flags、发现号码、隐藏线索、重复检查和条件显示。
- [x] 鼠标、触摸 Pointer Events 和键盘替代路径均可用。

### 剧情与重玩

- [x] 四幕主线覆盖日常电话、话术入侵、网络显形和交换机反制。
- [x] 天气、Meridian、内部席位、总机、失物招领、午夜广播和紧急服务组成互相指向的电话网络。
- [x] 节点文本支持按会话种子和访问次数稳定变化的 variants。
- [x] 已发现号码、线索、结局和尝试次数跨周目保留。
- [x] 成功、失败、模糊与隐藏结局全部存在，主要线路均可挂断、超时或进入结局。

### 事件图与后台

- [x] 图编辑器可创建、移动、连接、选择和删除节点 / 转场并保存坐标。
- [x] 节点 Inspector 可编辑类型、标签、文本 variants、fallback、备注、说话者、LCD、自动推进、失真、挂断权限和结局。
- [x] 转场 Inspector 可编辑所有触发类型、样例事件、选择文案 / 语气 / 隐藏状态、优先级、conditions 和 effects。
- [x] 支持 `dialNumber`、`choice`、`incomingAnswer`、`hangUp`、`timeout`、`sceneInspect`、`keywordAny` 和 `auto`。
- [x] 号码与超时 Inspector 可编辑空号 / 忙音节点、紧急号码、4 类超时、有效号码簿和主动来电排程。
- [x] 热点 Inspector 可编辑无障碍描述、首次 / 重复文案、发现号码、显示条件和百分比区域。
- [x] JSON 条件编辑器允许保留草稿，失焦解析失败时标记无效而不覆盖现有数据。
- [x] 运行预览显示当前节点、LCD、回合、结局、flags、命中边、候选边和模拟通话记录。
- [x] 支持 JSON 导入 / 导出；开发服务器写回源码，静态环境回退到 `localStorage`。

### 音画与记录

- [x] 固定 2D 电话亭场景包含多层雨幕、玻璃反光、红色框架、纸张热点、电话机与氛围干扰。
- [x] Web Audio API 实时生成雨声、铃声、拨号音、忙音、空号提示、线路噪声、低语、转盘棘轮、回弹和听筒反馈。
- [x] 音频在首次交互后解锁，可随时静音；通话结束只停止线路音并保留雨夜氛围。
- [x] 结局页可进入档案；档案包含会话编号、日期、结局、完整双方转录、拨号记录、线索、发现号码与结局索引。
- [x] 最近 16 次结局被归档，并可清除全部本地进度。
- [x] 提供生成式制作的专属社交分享图 `public/og.png`，页面已配置 Open Graph 与 Twitter Card 元数据。
